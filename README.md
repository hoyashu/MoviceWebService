# Marge 전략
## Branch Naming
작업 브랜치명 : `dev`
작업 단위 브랜치명 : `feat-xx` (ex.feat-01)
최종 브랜치명 : `master`

## Branch Operation
`dev`에서 작업 중, 단위별로 `feat`Branch를 생성한다. <br>
해당 Branch를 인호님께 PR받고 `Commit Marge`한다.

**규칙은 꼭 지킬 것!**

Open [http://localhost:3000](http://localhost:3000) to view it in the browser.
